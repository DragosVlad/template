# template
Default HTML Template

Instructions for setup:

  GIT
    
    Initialisation:
      - Create template folder
      - git init
      - git remote add template https://github.com/DragosVlad/template.git
      - git pull template
    
    Work:
      - Make changes
      - git add .
      - git commit -m "your message here"
      - git push template
